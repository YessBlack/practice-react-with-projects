# Rick and Morty Characters

![rick-and-morty](https://media.giphy.com/media/l378BzHA5FwWFXVSg/giphy.gif)
Este proyecto es una aplicación web que muestra los personajes de la serie animada Rick and Morty. La aplicación utiliza React como librería de interfaz de usuario, Vite como herramienta de construcción y Infinite Scroll para cargar los personajes dinámicamente a medida que se hace scroll en la página.

## Instalación y configuración

1.  Clona este repositorio en tu equipo:

`https://github.com/YessBlack/practice-react-with-projects/tree/main/06-rick-and-morty` 

2.  Navega hasta el directorio del proyecto:

`cd 06-rick-and-morty` 

3.  Instala las dependencias:

`npm install` 

4.  Inicia la aplicación:

`npm run dev` 

5.  Abre la aplicación en tu navegador en [http://localhost:5173/](http://localhost:5173/).

## Uso

La aplicación muestra los personajes de Rick and Morty en una lista infinita que se carga dinámicamente a medida que se hace scroll en la página.
